Tímaflækja

tvenns konar. Áætluð tímaflækja út frá kóða og hin, mæld tímaflækja. Hægt að plotta upp á grafi. þÁ sjáum við grafið sem tímaflækju

int sum = 0;
for (int i = 0; i < n; i++0
  for(int j = 1; j < n; j = j*2)
    sum = sum + i * j;

svar: T(n) ~n lg n
